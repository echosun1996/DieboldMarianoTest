## 特点
通过DM检验的结果，可以判别任意两类算法的预测结果是否存在差异性。

## 安装

```bash
pip install diebold-mariano-test
```

## 更新

```bash
pip install --upgrade diebold-mariano-test
```

## 使用说明

```bash
import diebold_mariano_test
```
